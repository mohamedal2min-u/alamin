### [M-00] منصة — Platform

**المسؤول:** Super Admin  
**العزل:** لا يوجد farm_id (بيانات عامة للمنصة)

| Entity | الوصف |
|---|---|
| `Plan` | خطط الاشتراك (حدود القطعان، المستخدمين، الأصناف) |
| `Farm` | المزارع المشتركة في المنصة |

**Screens (Next.js — Super Admin Panel):**
- `/super/dashboard` — إحصاءات المنصة
- `/super/farms` — قائمة المزارع + الحالة
- `/super/plans` — إدارة خطط الاشتراك

### [M-01] المصادقة — Auth

**التقنية:** Laravel Sanctum (API Token)  
**القاعدة:** Token يحمل `farm_ids` المسموح بها للمستخدم

| العملية | Method | Endpoint |
|---|---|---|
| تسجيل الدخول | POST | `/api/auth/login` |
| تسجيل الخروج | POST | `/api/auth/logout` |
| بيانات المستخدم | GET | `/api/auth/me` |

**Screens:**
- `/login` — صفحة الدخول (عربي RTL)

---

### [M-02] إعدادات المزرعة — Farm Settings

**العزل:** `farm_id` إلزامي  
**الصلاحيات (Spatie):** `farm_admin` فقط يمكنه التعديل

| Entity | الوصف |
|---|---|
| `Farm` | اسم، موقع، العملة (USD ثابت) |
| `Shed` | العنابر: اسم، مساحة، سعة |
| `FarmUser` | ربط مستخدم بمزرعة + دور Spatie |

**Business Rules:**
- لا يمكن حذف عنبر يحتوي قطيعًا نشطًا
- كل مستخدم يمكن أن يكون في أكثر من مزرعة بأدوار مختلفة

**Screens:**
- `/farm/settings` — بيانات المزرعة
- `/farm/sheds` — إدارة العنابر
- `/farm/users` — إدارة الأدوار

### [M-03] القطعان — Flocks ★ (مركز التشغيل)

**العزل:** `farm_id` إلزامي  
**الحالات:** `active → harvesting → closed → archived`

| Entity | الوصف |
|---|---|
| `Flock` | القطيع: العنبر، العدد، تاريخ البداية، السلالة |
| `FlockDailyRecord` | يومي: نفوق، وزن، علف، درجة حرارة |
| `Harvest` | حصاد: عدد الطيور، الوزن، السعر |

**Business Rules:**
- عنبر واحد = قطيع نشط واحد فقط
- السجل اليومي = يوم واحد فقط لكل قطيع (UNIQUE)
- `current_count = initial_count - SUM(mortality) - SUM(culled) - SUM(harvest_birds)`
- الحصاد النهائي (`final`) يغلق الدورة تلقائيًا
- FCR = إجمالي العلف المستهلك (كغ) ÷ إجمالي الوزن المحصود (كغ)
- تكلفة الكيلو = إجمالي مصاريف الدورة ÷ إجمالي الوزن المحصود

**Screens:**
- `/farm/flocks` — قائمة القطعان
- `/farm/flocks/new` — إنشاء قطيع
- `/farm/flocks/[id]` — تفاصيل القطيع
- `/farm/flocks/[id]/daily` — إدخال السجل اليومي
- `/farm/flocks/[id]/harvest` — تسجيل حصاد

### [M-04] المخزون — Inventory

**العزل:** `farm_id` إلزامي  
**التصنيفات:** أعلاف، أدوية، مطاعيم، مستلزمات، وقود، تنظيف

| Entity | الوصف |
|---|---|
| `ItemCategory` | Lookup — تصنيفات ثابتة للمنصة |
| `InventoryItem` | الصنف: اسم، وحدة، رصيد حالي |
| `StockMovement` | حركة: نوع، كمية، مرجع (flock/عام) |

**Business Rules:**
- كل استهلاك علف مرتبط بـ `flock_id`
- لا يمكن إخراج كمية أكبر من الرصيد الحالي
- تنبيه تلقائي عند وصول الرصيد لأقل من `min_stock_level`
- الرصيد يُحسب: `SUM(in) - SUM(out)` من `stock_movements`

**أنواع حركات المخزون:**
```
purchase   → شراء (in)
consumption → استهلاك (out) — مرتبط بـ flock
transfer   → نقل بين عنابر
adjustment → تصحيح جرد
return     → إرجاع لمورد (in)
waste      → هدر (out)
```

**Screens:**
- `/farm/inventory` — قائمة الأصناف + تنبيهات
- `/farm/inventory/[id]` — تاريخ حركات الصنف
- `/farm/inventory/movements/new` — تسجيل حركة

### [M-05] المبيعات — Sales

**العزل:** `farm_id` إلزامي  
**العملة:** USD ثابت

| Entity | الوصف |
|---|---|
| `Customer` | العملاء: اسم، هاتف، نوع |
| `Sale` | الفاتورة: رقم، تاريخ، إجماليات، حالة الدفع |
| `SaleItem` | بنود الفاتورة: نوع المنتج، كمية، سعر |

**أنواع المنتجات في الفاتورة:**
- `live_bird` — دجاج حي (الحالة الأكثر شيوعًا)
- `egg` — بيض (للقطعان البيّاضة مستقبلاً)
- `slaughtered` — مذبوح
- `byproduct` — منتج فرعي (ريش، سماد)

**Business Rules:**
- كل حصاد مرتبط تلقائيًا بفاتورة بيع مقابلة
- الفاتورة = الإجمالي + الخصم + الضريبة (إن وجدت)
- `payment_status`: pending → partial → paid

**Screens:**
- `/farm/sales` — قائمة الفواتير
- `/farm/sales/new` — فاتورة جديدة
- `/farm/customers` — إدارة العملاء

---

### [M-06] المصاريف — Expenses

**العزل:** `farm_id` إلزامي  
**التصنيفات:** عمالة، أعلاف، أدوية، صيانة، كهرباء/مياه، وقود/نقل، إيجار، أخرى

| Entity | الوصف |
|---|---|
| `ExpenseCategory` | Lookup — تصنيفات ثابتة |
| `Expense` | مصروف: المبلغ، التاريخ، القطيع المرتبط، الوصف |

**Business Rules:**
- المصروف قد يكون عامًا (مزرعة) أو خاصًا بقطيع (`flock_id`)
- مصاريف القطيع تدخل في حساب تكلفة الدورة
- لا يمكن حذف مصروف معتمد — يمكن إلغاؤه فقط

**Screens:**
- `/farm/expenses` — قائمة المصاريف
- `/farm/expenses/new` — إضافة مصروف

---

### [M-07] العمال — Workers

**العزل:** `farm_id` إلزامي

| Entity | الوصف |
|---|---|
| `Worker` | بيانات العامل: اسم، هاتف، الراتب، العنبر |
| `WorkerTask` | مهمة يومية: النوع، التاريخ، القطيع المرتبط |

**Business Rules:**
- العامل قد يكون مرتبطًا بحساب مستخدم (`user_id`) أو مجرد سجل
- الراتب: شهري، يومي، أو حسب المهمة
- مصروف الراتب يُنشأ تلقائيًا في نهاية الشهر

**Screens:**
- `/farm/workers` — قائمة العمال
- `/farm/workers/[id]` — ملف العامل + مهامه

---

### [M-08] التقارير — Reports

**العزل:** `farm_id` إلزامي  
**الصلاحيات:** `farm_admin` و `farm_manager`

| التقرير | الوصف | المصدر |
|---|---|---|
| تقرير الدورة | ملخص كامل لقطيع منتهٍ | flocks + expenses + harvests |
| تقرير FCR | معدل تحويل العلف | flock_daily_records |
| تكلفة الكيلو | الإجمالي ÷ الوزن | expenses + harvests |
| الربح والخسارة | إيرادات - مصاريف | sales + expenses |
| مقارنة الدورات | مقارنة بين قطعانين | flocks متعددة |
| المخزون | الرصيد والحركات | stock_movements |

**Screens:**
- `/farm/reports/cycle/[flock_id]` — تقرير دورة
- `/farm/reports/comparison` — مقارنة دورات
